# TCC – GEPARK
(Que Deus nos ajude)



📦 Instalação do Projeto
```bash
cd front-GEPARK
nvm install 22
npm install
npm run dev
```



⚙️ Variáveis de Ambiente
Crie um arquivo **.env** na raiz do projeto:
VITE_API_URL=https://api.openweathermap.org/data/2.5
VITE_API_KEY=SEU_API_KEY_AQUI
