# TCC – BACK GEPARK



**Instalação do Projeto**
```bash
cd back-GEPARK
npm install
node scripts/seedIbge.js
npm run dev
```
Acessar: http://localhost/phpmyadmin


**Variáveis de Ambiente**
Crie um arquivo **.env** na raiz do projeto:
PORT=3001

DB_HOST=localhost
DB_USER=root
DB_PASSWORD=
DB_NAME=gepark

JWT_SECRET=gepark_secret_key_2026_@TCC
